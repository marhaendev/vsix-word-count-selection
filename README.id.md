# Word Count Selection

**Word Count Selection** adalah ekstensi VS Code sederhana dan efisien yang menampilkan jumlah kata dan karakter dari teks yang Anda pilih secara **real-time** di Status Bar.

## Fitur Utama

- **Real-time Counting**: Hitungan otomatis diperbarui segera setelah Anda memblok teks.
- **Status Bar Integration**: Tampil rapi di pojok kanan bawah (sebelah info baris/kolom) hanya ketika ada teks yang dipilih.
- **Clean UI**: Otomatis tersembunyi jika tidak ada teks yang dipilih, menjaga tampilan editor tetap bersih.
- **Word & Char Count**: Menampilkan jumlah kata (Words) dan karakter (Chars) sekaligus.

## Cara Penggunaan

1.  Instal ekstensi ini.
2.  Buka file teks apa saja di VS Code.
3.  **Blok / Seleksi** teks yang ingin dihitung.
4.  Lihat di **Status Bar** (pojok kanan bawah), akan muncul informasi seperti:
    `Word Count: 5 Words, 25 Chars`

## Pengaturan Ekstensi

Ekstensi ini bekerja secara otomatis tanpa konfigurasi khusus. Aktivasi terjadi segera setelah Anda membuka VS Code (`*`).
